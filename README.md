Менеджер задач

Менеджер задач - десктопное приложение, позволяющее создавать задачи.
К задачам можно добавлять субзадачи, к которым уже привязан прогресс.
Прогрессом управляет пользователь: добавить/убавить прогресс субзадачи, завершить/начать задачу/субзадачу.
Все представлено в удобном GUI-Интерфейсе, позволяющем пользователю очень легко управлять своими задачами.

Функции:
1. Добавление задач
2. Добавление субзадач
3. Добавление тегов
4. Добавление "крайнего срока", при достижении которого задача будет просрочена, а выполнять её станет невозможно
5. Добавление напоминания, при дате, достижении которой пользователю будет высвечено уведомление-напоминание

Дополнительно:
1. Удобный GUI-Интерфейс
2. Локальное сохранение данных
